Proxy & Browser

Proxy

Proxies operate the unblocking backend, enhancing your access to games and apps. Adjusting the proxy can improve game performance, speed up site loading, and enhance your overall experience.

Transport

Transport refers to how the proxy transmits information. While changing the transport can improve the performance of some proxies, it might cause issues for others. This feature is advanced and not recommended for those unfamiliar with it.

Search Engine

The Search Engine Selector allows you to choose from multiple search engines, offering customizable options and performance insights for an improved search experience.

Save Data
